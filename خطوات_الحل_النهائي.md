# خطوات حل مشكلة تسجيل الدخول - نهائياً

## الخطوة 1: تشغيل السكريبت في Supabase

1. افتح مشروعك في Supabase
2. اذهب إلى **SQL Editor** من القائمة الجانبية
3. اضغط **New Query**
4. انسخ محتوى ملف `scripts/fix-database-final.sql` بالكامل
5. الصقه في المحرر
6. اضغط **Run** أو **F5**
7. انتظر حتى تظهر رسالة Success

## الخطوة 2: تأكد من نجاح العملية

في نفس SQL Editor، شغّل هذا الأمر:

```sql
SELECT * FROM users;
```

يجب أن ترى مستخدم واحد:
- username: admin
- password: admin
- role: admin
- is_active: true

## الخطوة 3: سجل الدخول

1. ارجع للتطبيق
2. اسم المستخدم: **admin**
3. كلمة المرور: **admin**
4. اضغط تسجيل الدخول

## إذا استمرت المشكلة:

افتح Console المتصفح (F12) وابحث عن:
- أي رسائل خطأ حمراء
- رسائل تبدأ بـ [v0]
- رسائل تحتوي على "login" أو "database"

وأرسل لي هذه الرسائل.

---

## ملاحظة مهمة:

هذا السكريبت يحذف **جميع البيانات القديمة** ويعيد إنشاء قاعدة البيانات من الصفر.
إذا كان لديك بيانات مهمة، أخذ نسخة احتياطية أولاً.

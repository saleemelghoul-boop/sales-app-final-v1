# حل مشكلة تسجيل الدخول

## المشكلة
لا يمكن تسجيل الدخول باسم admin وكلمة المرور admin

## الحل السريع

### الخطوة 1: تشغيل سكريبت SQL في Supabase

1. افتح Supabase واذهب إلى SQL Editor
2. انسخ والصق هذا الكود:

```sql
DELETE FROM users WHERE username = 'admin';

INSERT INTO users (username, password, full_name, role, is_active, admin_permission)
VALUES ('admin', 'admin', 'المدير العام', 'admin', true, 'full');

SELECT * FROM users WHERE username = 'admin';
```

3. اضغط Run
4. يجب أن تظهر نتيجة تحتوي على المستخدم admin

### الخطوة 2: تسجيل الدخول

الآن يمكنك تسجيل الدخول:
- اسم المستخدم: admin
- كلمة المرور: admin

## التحقق من المشكلة

إذا استمرت المشكلة:

1. افتح Console المتصفح (F12)
2. حاول تسجيل الدخول
3. ابحث عن رسائل `[v0]` في Console
4. ستظهر لك رسائل توضح المشكلة بالضبط

|-------|--------------|-------------|
| مدير | admin | admin |
| مندوب تجريبي | rep1 | rep1 |
